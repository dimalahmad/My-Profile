# Assets Directory

## Profile Image

To add your profile photo:

1. Add your profile image as `profile.jpg` in this directory (`public/assets/profile.jpg`)
2. Recommended dimensions: 320x320 pixels (square)
3. Supported formats: JPG, PNG, WebP

If no image is provided, the Hero component will automatically use a generated avatar with your name.

## Other Assets

You can add other assets like:
- Project screenshots
- Certificate images
- Icons
- Other media files

All files in the `public` directory are served statically and can be referenced with `/assets/filename.ext` in your components.
